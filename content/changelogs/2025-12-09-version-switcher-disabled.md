The launcher's version switcher has been disabled for now due to various reliability issues:

- Attempting to downgrade from a GDK version to a UWP version simply fails
  - This still occurs when attempting to directly install the game's app package
- The legacy version switcher doesn't work correctly due to the changes brought in by the GDK

**For switching versions, please use the Bedrock Launcher, Jiayi Launcher, or any tool that allows UWP versions to be installed.**

## Changes üîé

- The general page within the launcher's settings has been slightly revamped

- Auto-updates for the launcher are no longer forced - you can now postpone the update
