# MINOR LAUNCHER UPDATE 游

## Changes 游늾

- License verification is now represented by a colored emoji beside the launcher version text:
  - 游댮 - Unverified game ownership
  - 游릭 - Verified game ownership
  - 游리 - Cannot verify game ownership

- The game version text will now change colors representing whether or not a game version is supported by Flarial:
  - 游리 - Unsupported game version
  - 游릭 - Supported game version

- Improved the "Wait For Resources" feature
