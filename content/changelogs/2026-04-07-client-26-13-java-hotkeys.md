# CLIENT UPDATE

## Changes

- Future-proofed the client for `1.26.13` (let us know if anything goes wrong).
- Fixed tab complete for client commands.
- Added the `Java Hotkeys Fix` module — fixes hotkeys with broken or old resource packs, supports extra binds, and can override vanilla keybinds.

> **Note:** At the time of writing, the launcher does not yet support `1.26.13`. See pinned attachments for a workaround.

Please report crashes in `community-support` / `newcomer-support`.
