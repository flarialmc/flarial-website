# MINOR LAUNCHER UPDATE

## Changes

- Fixed a lot of scheduling issues with the launcher's frontend.
  - This should resolve undesired behaviors in the frontend.
- The `Beta` DLL isn't auto-enabled if a GDK build is detected — you must enable it manually.
- The launcher's version switcher now exposes GDK builds.
- All available GDK builds will be available until Flarial supports GDK builds fully.
- You can now easily upgrade to the latest GDK build(s) without the Microsoft Store!

Please let us know if there are any issues via our [GitHub Issue Tracker](https://github.com/flarialmc/dll/issues) or community support.
