# CLIENT UPDATE

## Changes

- Added beta support for Minecraft Bedrock `1.26.20`.
- A lot of modules have been disabled for this update — we're actively working to bring them back.

> **Heads-up:** If your game keeps crashing on the latest version, revert to a previous Bedrock version. This release is extremely buggy on Mojang's end.

Please report any bugs and issues in `community-support` or `newcomer-support` — we can't fix issues if we don't know they exist.
