# MINOR LAUNCHER UPDATE

## Changes

- Added support for `1.21.13X`.
- Improved performance of the `Versions` page.
- Improved the `Versions` page progress indicator.
- Improved version list filtering — the `Versions` page now only displays supported versions.
- The latest version lists are now fetched instead of cached for jsDelivr links.

Please let us know if there are any issues via our [GitHub Issue Tracker](https://github.com/flarialmc/dll/issues) or community support.
