# CLIENT UPDATE

## Additions

- Added `Shader Loader` — loads BRD-compatible shaders.
- Added configurable sensitivity for horizontal and vertical axes (inside the `Sens Multiplier` module).
- Added `Crystal Optimizer`.
- Added Multi Keybind support for Minecraft keybinds (inside the `Modern Handling` module).
- Added support for a custom crosshair when looking at an entity or block.
- Added support for changing the hitbox color of an entity you are looking at.

## Bug Fixes

- Fixed a bug related to keybinds not being able to be changed on new configs.

Please report any bugs and issues in `community-support` or `newcomer-support` — we can't fix issues if we don't know they exist.
