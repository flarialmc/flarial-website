# CLIENT UPDATE — Very Big Update

## New / Improved

- Added an `Item Tracker` feed for item pickups/drops in a subtitle-style format.
- Added a `Durability Notifier`.
- Improved the `Locator Bar`.
- Improved Realm compatibility.
- Ported over `Mouse Tweaks` (the Java Edition mod) — only works in singleplayer.
- Improved `Item Counter` behavior, scaling, alignment, and tracking stability.
- `ClickGUI` no longer blurs the background while you're in module settings.
- Added Discord Presence.
  - Bonus: Hive gamemode names show as well.
- Added more `Hive Stat` table customization.
- Enhanced `Hive Stats`:
  - Better leaderboard presentation.
  - Highlighting player team colors.
- Improved the `Potion HUD` with more customization options.
- Added tab list player highlighting.
- Added dynamic lighting — the feature you knew and loved from OptiFine / LambDynamicLights.
- Improved the client commands system with autocomplete suggestions.
- Re-added `Render Options`.
- Enhanced the Flarial watermark in GUIs.

## Fixes

- Fixed some client crash scenarios.
- Improved crash logs with more useful context.
- Fixed slider input crashes from invalid number parsing.
- Fixed scripting HTTP request hangs.
- Fixed rotation snapping/jumps across yaw boundaries.
- Fixed `player.say()`.
- Fixed a `Player Notifier` crash.
- Fixed the `Custom Crosshair` editor not opening.

Please report any bugs and issues in `community-support` or `newcomer-support` — we can't fix issues if we don't know they exist. Thanks to the community feedback and suggestions that helped with this update.
